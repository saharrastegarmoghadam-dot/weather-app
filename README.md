# 🌤️ Weather App

## 📌 Description

This project is a simple Weather Application built using Python and the OpenWeather API. Users can enter a city name and receive real-time weather information including temperature and weather conditions.

## 🛠️ Tools Used

- Python
- Requests
- JSON
- OpenWeather API

## 📁 Project Structure

```text
weather-app/
│
├── main.py
└── README.md
```

## 📊 Features

- Search weather by city
- Retrieve real-time weather data
- Display temperature
- Display weather conditions
- Handle invalid city names

## 🚀 How to Run

1. Install required package:

```bash
pip install requests
```

2. Get a free API key from OpenWeather.

3. Add your API key in `main.py`.

4. Run the application:

```bash
python3 main.py
```
